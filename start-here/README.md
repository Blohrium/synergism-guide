## Reset Order

Prestige → Transcend → Reincarnate. Each unlocks permanently on first use.

Coin milestones that reveal new buildings: 500, 10,000, 100,000, 4,000,000.

## Challenge Order

- c8x1 — unlocks the Anthill (required for Galactic Crumbs)
- c9x1 — unlocks Talismans and Blessings
- c10x1 — unlocks Ascensions

These gate each other. You need c8 before you can meaningfully build toward c10.

## Challenge 10 - Sadistic Challenge I

Combines all five transcension challenge restrictions simultaneously:

- Accelerator Power = 1
- Multiplier Power = 1
- Building costs spike super-exponentially after 25,000 purchased (affected by Building Cost Delay)
- Diamond gain = Coins^0.0001
- Mythos gain = Coins^0.001

**Goal:** reach the coin requirement. Base exponent is 2.5e9 (scales each completion).

**First completion reward:** Ascensions unlocked.

**Per-completion rewards:** +100 base ant ELO, +2% Ant Sacrifice Reward, +10% Offerings.

## Preparing for c10

**What carries you inside c10:** Galactic Crumbs (multiply ant speed logarithmically) and ant ELO. Accelerators and Multipliers do nothing — ants and base building power are your production.

~1e300 Galactic Crumbs recommended before attempting (from in-game flavor text).

**Requirement reductions** — each reduces the base exponent by 1e8, lowering the coin target:
- r6x15, r7x5, r7x20, r8x10
- Shop upgrades: Challenge Tome 1 and Challenge Tome 2
