# Start Here - c10x1

**Goal:** Complete Challenge 10 (Sadistic Challenge I) once to unlock Ascensions.

## Reset Order

Prestige → Transcend → Reincarnate. Each unlocks permanently on first use.

## Challenge Order

- c8x1 — unlocks the Anthill (Galactic Crumbs)
- c9x1 — unlocks Talismans and Blessings
- c10x1 — unlocks Ascensions

## Challenge 10 - Sadistic Challenge I

Combines all five transcension challenge restrictions simultaneously:

- Accelerator Power = 1
- Multiplier Power = 1
- Building costs spike super-exponentially after 25,000 purchased
- Diamond gain = Coins^0.0001
- Mythos gain = Coins^0.001

**Goal:** reach the coin requirement (base exponent 2.5e9, scales each completion).

**What carries you:** Galactic Crumbs and ant ELO. ~1e300 Galactic Crumbs recommended before attempting.

**Requirement reductions** (each cuts the base exponent by 1e8):
- r6x15, r7x5, r7x20, r8x10
- Shop: Challenge Tome 1 and Challenge Tome 2

**First completion reward:** Ascensions unlocked.
