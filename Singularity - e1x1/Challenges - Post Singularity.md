## Content

### c15

c15 expo (best Challenge 15 exponent) is a key progression stat post-singularity. Run c15 with an Off loadout, ideally including an RCM module. After AoAG penalties are removed, c15 gives significantly more exponent, which gives more quarks.

Occasionally check if a better c15 exponent is achievable pre-AoAG if within c6 — waiting a few seconds without auto-challenge can let the exponent rise slightly.

After AoAG, c15 spam becomes viable: with auto ascend on and c15 as the only active ascension challenge, the game spams c15 automatically, producing massive exponent after Omega.

### c10

Required for ascension count gain in e3. In e3, exit c10 manually after each completion to trigger the next ascension challenge — auto challenge will waste ascensions if left on.

### Corruption-Era Challenge Loadouts

For challenge loadouts from c10x1 through post-singularity, see [Corruption Glossary](Corruption%20Glossary.md).

### e2 Challenge Cap

In e2, a maximum of one completion each from c1–c14 is allowed (the Not So Challenging perk adds 5 c8 completions bypassing this). Use the highest corruption levels clearable while still completing c10; raise Viscosity last.

## FAQ

```
Q: Why does c15 matter post-singularity?
A: c15 expo determines quark gain post-AoAG. After penalties are removed you gain significantly more exponent, meaning more quarks.
```

```
Q: What loadout should I use for c15?
A: Off loadout with an RCM module where possible.
```

```
Q: Why can't I gain ascension count in e3?
A: Ascending without completing c10 doesn't grant ascension count in e3. Turn off auto challenge and manually complete c10 each time.
```
