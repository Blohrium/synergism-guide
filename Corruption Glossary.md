## Content

> Corruption loadouts are listed top to bottom as they appear in the corruption screen. This is **not** the import order. Import order differs — note this before importing any loadout string.

Loadout notation:
- `x/y/…/z` — corruption levels in screen order (top to bottom)
- `/c` — run optimised for 3D cubes
- `/s` — run optimised for score
- `/o` — run optimised for octeracts (5D+ cubes depending on stage)
- `M` — maximum possible corruption level available at that stage

Keep ~8 loadout slots saved (or ~16 with pseudo coin upgrades) to avoid consulting this page every run.

## FAQ

```
Q: Are these loadouts in import order?
A: No. All loadouts are listed top to bottom as shown in the corruption screen. Import order is different — do not import these strings directly without checking the import order first.
```

```
Q: What does M mean in a loadout?
A: Maximum possible corruption level at that stage.
```

```
Q: How many loadout slots should I keep saved?
A: Around 8. If you have pseudo coin upgrades, around 16.
```

```
Q: What is the difference between /c, /s, and /o runs?
A: /c = optimised for 3D cubes. /s = optimised for score. /o = optimised for octeracts (5D+ cubes depending on stage).
```
